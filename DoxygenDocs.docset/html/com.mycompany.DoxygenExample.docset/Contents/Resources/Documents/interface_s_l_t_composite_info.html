Detailed Description

This is the interface of SLTCompositeInfo class.

This class holds the information of composite asset, which is being extracted from a corresponding composite of the "composites" node of JSON:

"boards": { "main": { "composites": [ { "assetId": 2839, "position": [0, 10] }, .... ], "chunks": [...], "blockedCells": ... "properties": { "cell" : [{...}] } } } ... "assets": { "2839": { "cells": [[0, 0],..., [0, 10], [1, 16]] "keys": { "COLOR": 1, "CARD_SUIT": 1, "CARD_VALUE": 1 }, "states": [ 534 ], "type_key": "normal"

}, }
