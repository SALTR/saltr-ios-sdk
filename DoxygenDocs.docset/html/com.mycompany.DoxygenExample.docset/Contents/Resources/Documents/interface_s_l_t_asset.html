Detailed Description

This is interface for SLTAsset class, that is being used internally only by SDK.

This is the base class to store the basic data of asset from parsed JSON.

As an example, the asset data from the JSON looks like :

{ ... "assets": { "2835": { "keys": { "COLOR": 1, "CARD_SUIT": 1, "CARD_VALUE": 1 }, "states": [ 534 ], "type_key": "normal"

}, } }...
