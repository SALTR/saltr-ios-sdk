Detailed Description

This is public interface of SLTAssetInstance class.

An instance of this class holds basic asset data (type, keys) and also value of state.

As an example, asset data with state info from the JSON looks like : { "assetStates": { "534": "state1", "535": "state2" }, ... "boards": { "main": { ... } } "assets": { "2835": { "keys": { "COLOR": 1, "CARD_SUIT": 1, "CARD_VALUE": 1 }, "states": [ 534, 535

], "type_key": "normal"

}, } }
